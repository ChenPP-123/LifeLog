# LifeLog

一个使用 Python 编写的终端命令行日志与任务管理工具。

## Features

- Task Management
    - add task
    - list task
    - mark task done
    - delete task
- Log system
    - add log
    - show existing logs

## Supplementary Explanation

- All data is kept in a json file.
- Conforms to the usage habits of major terminal tools
